# Portfolio_ADi-RED

A visually immersive and responsive personal portfolio website built using HTML, CSS, and JavaScript. This project presents my professional identity as a UI/UX Designer and SEO Content Writer, combining design aesthetics with functional front-end development.

## 📌 Live Preview  
🔗 https://atiqueasif2.netlify.app/

## 📖 About The Project  

This portfolio website is designed to communicate my skills, projects, and services through a structured and engaging user experience.  

The design follows a **modern dark-themed interface enhanced with motion and interaction**, aiming to create a strong visual impact while maintaining clarity and usability.  

The project reflects my approach to **user-centered design, content clarity, and front-end execution**, where both visual design and interaction logic work together.

---

## ✨ Key Features  

- 🎥 **Fullscreen Background Video**  
  Creates an immersive first impression and strengthens visual identity  

- 🧭 **Responsive Navigation System**  
  Includes desktop navigation and a mobile hamburger menu with smooth open/close behavior  

- 🎯 **Smooth Scrolling with Offset Control**  
  Ensures accurate section positioning considering fixed header  

- 📍 **Active Section Highlighting**  
  Navigation links dynamically update based on scroll position  

- 🎬 **Scroll-Based Animations (AOS)**  
  Elements animate into view to improve engagement and visual flow  

- 🧩 **Modular Section Structure**  
  Organized sections: Hero, About, Skills, Projects, Contact  

- 🧠 **Interactive UI Components**  
  - Hover animations on cards  
  - Animated skill grid  
  - CTA buttons with scroll triggers  

- 📱 **Fully Responsive Layout**  
  Optimized for mobile, tablet, and desktop devices  

- 📩 **Contact Form with Basic Validation**  
  JavaScript-based validation ensures required fields are filled before submission

## ⚙️ Functional Overview  

The application is structured around **event-driven interactions and scroll-based behavior**.

- The **navigation system** handles both desktop and mobile views, including menu toggling and scroll navigation.  
- A **custom smooth scrolling mechanism** ensures consistent spacing and alignment with the fixed header.  
- The **scroll listener** dynamically updates active navigation states and triggers UI changes.  
- The **hero section** uses SVG-based animated typography combined with a call-to-action button.  
- The **projects section** presents content using reusable card components with hover-based interaction.  
- The **contact form** includes basic validation logic and user feedback via alert messages.
